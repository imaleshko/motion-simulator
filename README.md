# Симуляція рухів

Вебзастосунок для моделювання та 3D-візуалізації двох видів механічного руху:

- **Прямолінійний рівноприскорений рух** матеріальної точки;
- **Рух тіла, кинутого під кутом до горизонту** (з урахуванням опору повітря).

Користувач задає параметри у формі, а траєкторія в реальному часі відображається на інтерактивному тривимірному графіку.

## Можливості

- Перемикання між двома моделями руху.
- Розкладання початкового вектора швидкості за двома кутами — вертикальним θ і горизонтальним φ.
- Інтерактивний 3D-графік траєкторії (обертання, масштабування, наведення) на базі Plotly.
- Миттєвий перерахунок траєкторії при зміні будь-якого параметра.

## Стек

- **React** + **TypeScript**
- **Plotly.js** + **react-plotly.js**

## Швидкий старт

```bash
npm install

npm run dev
```
